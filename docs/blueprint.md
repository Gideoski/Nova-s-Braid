# **App Name**: Nova's Braid Bar

## Core Features:

- Service Catalog: Display available braiding services with categorized listings (Knotless, Twist, Fulani Braids, French Curls, Extra) and pricing.
- Booking System: Allow users to book appointments for individual or group sessions, selecting service types, dates, and times.
- Appointment Management: Enable users to view and manage their appointments.
- Automated WhatsApp Integration: Automatically forward client booking details and requests to admin's WhatsApp number after successful appointment booking.
- Settings: Allow the users to select their preferred theme for the application.
- Feedback Collection: Collect user feedback and send it to the admin's email for review and improvements.
- Contact Information Display: Show clickable contact information (phone number, WhatsApp, email) for easy communication with the salon.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a sleek, professional look.
- Background color: Light gray (#F0F0F0) for a clean backdrop.
- Accent color: Off-white (#E8E8E8) to highlight key elements.
- Body and headline font: 'Inter', a sans-serif font, will be used for both body text and headlines for a modern, clean aesthetic.
- Minimalist icons representing braiding services and salon features.
- Clean and professional layout with clear sections for services, booking, and contact information. Optimized for various devices.
- Subtle animations for transitions and loading screens to enhance user experience.
